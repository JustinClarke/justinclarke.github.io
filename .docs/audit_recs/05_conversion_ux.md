# 05: Conversion & UX Optimization

## 🔄 Internal Linking (SEO & Navigation)
Currently, project pages are "dead-ends." Adding cross-links between projects improves crawl depth and user engagement.

### Recommendation: "Related Case Studies"
Add a footer section to each project page with internal links using target keywords:
*   *"View the Data Architecture case study →"*
*   *"Explore the Machine Learning research →"*

## 🎯 CTA Refinement
### 1. "Return to Terminal"
Recruiters may not immediately understand what "Terminal" means in this context.
- **Visual**: Keep "Return to Terminal".
- **Aria-Label**: Use "Back to Portfolio Home".

### 2. Contact CTA
The "TheCloser" section is at the very bottom. For deep-linked recruiters, consider a sticky "Contact" pill at the top of the page (within the HUD status bar) to allow immediate conversion without scrolling.

## 📄 Social Validation
Ensure `og:image` is set. Each project should have its own unique Open Graph image (1200x630px) that showcases the project's visual identity.
- **LiteStore**: Blue-themed dashboard shot.
- **SQL Disaster**: Red-themed database schema shot.
- **HR Archetype**: White/System-themed diagnostic shot.
